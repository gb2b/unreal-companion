# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Hierarchical CLAUDE.md documentation (root + 5 local files)
- 21 Claude Code dev skills covering all project areas
- `frameworks/manifest.yaml` for template versioning
- Niagara tools documentation

### Changed
- Rewrote agent environment for better AI assistant onboarding
- Updated all IDE rule templates with correct tool count (87)
- Fixed broken agent → skill references in 6 agents
- Converted all documentation to English

### Removed
- `AGENTS.md` (absorbed into CLAUDE.md hierarchy)
- `.clinerules`, `.windsurfrules` (now generated by CLI at install)
- `.cursor/rules/companion/index.mdc` (now generated by CLI at install)
- Archived `PLAN.md` to `docs/archive/`

## [1.0.0] - 2025-01-01

### Added
- 87 MCP tools for Unreal Engine 5.7+ control (16 tool modules)
- C++ Plugin with TCP bridge (port 55557)
- Python MCP server with FastMCP and auto-discovery
- Web UI Studio (React 18 + FastAPI) on port 3179
- Interactive CLI (`npx unreal-companion`) with install, upgrade, workflow commands
- 9 AI agent personas (Zelda, Solid, Ada, Navi, Lara, Indie, Epic, Tester, Coach)
- 25 BMGD workflows across 6 phases
- 12 BMGD skills for game development
- Security token system with 4 risk levels
- Multi-agent chat interface with Party Mode
- Blueprint, actor, material, widget, landscape, niagara tools
- Batch operations with rollback support
